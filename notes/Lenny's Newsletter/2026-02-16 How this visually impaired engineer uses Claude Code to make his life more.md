# How this visually impaired engineer uses Claude Code to make his life more accessible | Joe McCormick

**Source**: [Lenny's Newsletter](https://www.lennysnewsletter.com/p/how-this-visually-impaired-engineer)
**Author**: Lenny Rachitsky (featuring Joe McCormick) | **Date**: Feb 16, 2026

---

## Summary

Joe McCormick, a visually impaired principal engineer, demonstrates how to use Claude Code to rapidly build custom Chrome extensions that address accessibility gaps overlooked by mainstream products, making personal software increasingly viable with AI assistance.

## Key Takeaways

- **Build accessibility tools quickly**: Create functional Chrome extensions in under 25 minutes using Claude Code, enabling you to solve specific accessibility problems without lengthy development cycles.
- **Leverage Claude Skills for patterns**: Use Claude Skills to identify and extract common development patterns across multiple extensions, reducing repetitive coding and accelerating future tool creation.
- **Design for keyboard-first workflows**: Implement keyboard shortcuts and screen reader accessibility techniques directly into custom tools to minimize cognitive friction and improve usability for assistive technology users.
- **Create personal software for niche needs**: Develop micro-extensions that address specific accessibility gaps in mainstream products (image descriptions in Slack, AI spell checkers, link summarization) that standardized tools don't provide.
- **Combine multimodal AI with accessibility**: Use Claude's multimodal capabilities alongside smart glasses and other hardware to create layered accessibility solutions that work together seamlessly.
