## Summary

Jesse Genet demonstrates how to build a practical multi-agent AI system using OpenClaw agents layered on Obsidian, with each agent assigned a specific role (homeschool, finance, scheduling, development, operations) running on dedicated hardware to automate household and business operations.

## Key Takeaways

- **Assign clear scope and responsibility** to each agent with distinct roles, personas, and SOUL.md files—treat agent onboarding like employee onboarding with defined responsibilities and decision-making boundaries.
- **Use decision files and incantations** as specialized prompting techniques to make agent behavior stick and predictable across multiple interactions and use cases.
- **Partition data across dedicated hardware** by running each agent on its own Mac Mini to improve performance, security, and cognitive focus rather than running all agents in a single environment.
- **Photograph physical inventory** to give agents real-world context—inventory photos of toys, books, and supplies enable AI to recommend actual materials during planning rather than generic suggestions.
- **Build hands-free workflows** by layering agents on top of existing systems (Obsidian, printing) to reduce friction—e.g., voice commands triggering print jobs instead of manual multi-step processes.
